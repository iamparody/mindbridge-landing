# MindBridge Landing Site

Standalone landing page for the MindBridge competition entry.

This project is intentionally separate from the main application in `src/`.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Notes

- No founder details or contact CTA are included yet.
- The page uses custom product-style visuals, not the old mockup screenshots.
- The copy avoids medical-service claims and keeps MindBridge positioned as a support platform.
